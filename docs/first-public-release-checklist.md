# First Public Release Checklist

## Repo polish

- [ ] README hero block updated
- [ ] description and topics set on GitHub
- [ ] docs folder reviewed for public clarity
- [ ] examples checked
- [ ] no private keys committed
- [ ] trust folder reviewed

## Technical checks

- [ ] npm install works cleanly
- [ ] npm run check passes
- [ ] npm run trust:all passes
- [ ] npm run release:all passes
- [ ] dist ZIP generated successfully

## Trust checks

- [ ] at least one trust bundle exported
- [ ] at least one replay manifest exported
- [ ] trust report generated
- [ ] release manifest generated
- [ ] public key exported to trust/public-keys

## Launch checks

- [ ] release notes reviewed
- [ ] announcement copy prepared
- [ ] first version tag chosen
- [ ] GitHub release drafted
- [ ] screenshots or diagrams prepared if desired
