# Promotion Model

## Goal

Not every memory is immediately trustworthy.

Operational Memory requires staged promotion.

## Flow

draft -> active -> verified

## Optional exits

active -> superseded
verified -> deprecated
* -> archived

## Meaning

- draft: captured, not yet approved
- active: usable in working context
- verified: safe to prioritize and reuse
- superseded: replaced by newer truth
- deprecated: retained for history, not recommended
- archived: inactive storage only
