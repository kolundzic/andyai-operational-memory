# Licensing Strategy

## Current License
This repository is licensed under **Apache-2.0**.

## Model
This repository follows a **HYBRID / OPEN-CORE** strategy:

- public core lives in this repository
- commercial moat is built above the core

## Public core
- schemas
- examples
- clients
- trust/replay foundations
- HTTP/API surfaces
- tests and verification

## Commercial moat above core
- hosted trust tooling
- enterprise governance packs
- premium adapters/exporters
- support/SLA
- managed validation and release layers
